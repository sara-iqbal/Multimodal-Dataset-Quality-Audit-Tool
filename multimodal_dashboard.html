Known Limitations
Alignment scoring is heuristic (no CLIP embeddings used in this version). Image quality based on metadata only — pixel-level analysis requires loading full images. Caption quality does not account for semantic accuracy.
Audit Pipeline
Stage 1 — Ingestion
3,000 image-caption pairs · Title + caption + image dimensions + format + mode
Stage 2 — Text Audit
Caption length · Char entropy · Word entropy · Repetition ratio · Flag detection
Stage 3 — Image Audit
Resolution · Aspect ratio · Format · Colour mode · Image quality score
Stage 4 — Alignment
Visual vocabulary overlap score · Caption descriptiveness · Length bonus
Stage 5 — Tier Assignment
Composite score · Flag-based removal · High / Medium / Low / Flagged
